IntentFlow 意图轨迹 - 完整使用指南

版本: 1.5.8 | 适用人群: 首次使用的用户 | 阅读时间: 约 10 分钟

🎯 这款插件能帮你做什么？

你是否遇到过这些问题？

  • 浏览器开了几十个标签页，完全不知道自己在干什么
  • 看着看着就跑偏了，忘记最初要查什么
  • 想回顾刚才的搜索过程，却想不起来访问了哪些网页

IntentFlow 的解决方案

这款插件会自动记录你访问的每个网页，并用 AI 分析这些网页的目的，帮你生成一个清晰的任务树。

简单来说：它像一个智能助手，帮你记录 "我刚才在网上做了什么"。

🚀 第一步：安装插件

1.1 打开 Chrome 扩展管理页面

  1. 打开 Chrome 浏览器
  2. 在地址栏输入：chrome://extensions
  3. 按回车键

1.2 开启开发者模式

  1. 在页面右上角，找到「开发者模式」开关
  2. 点击开关，使其变为蓝色/开启状态

1.3 加载插件

  1. 点击左上角的「加载已解压的扩展程序」按钮
  2. 在弹出的文件夹选择窗口中，找到 dist 文件夹
  3. 点击「选择文件夹」

1.4 确认安装成功

  • 浏览器右上角工具栏会出现插件图标（彩色圆点连线图案）
  • 点击图标，侧边栏会弹出

🎬 第二步：首次设置（重要！）

首次打开插件时，会弹出一个设置向导，请务必认真填写：

2.1 选择你的职业/角色

为什么要填这个？
不同职业的人访问同一个网站的目的可能完全不同。比如：

  • 程序员访问 GitHub = 工作/学习
  • 产品经理访问 GitHub = 竞品调研
  • 学生访问 GitHub = 学习编程

请选择最符合你的选项，例如：

  • 开发者 (Developer)
  • 产品经理 (Product Manager)
  • 设计师 (Designer)
  • 学生/研究员 (Student/Researcher)
  • 自由职业者 (Freelancer)
  • 等等...

2.2 描述你的典型使用场景

示例填写：

  • "日常工作中查找技术资料和文档"
  • "研究市场和竞争对手"
  • "学习新技能和在线课程"

填写完成后点击「下一步」→「完成」。

🖥️ 第三步：认识主界面

插件打开后，你会看到一个侧边面板。让我们逐一认识每个部分：

3.1 顶部工具栏（从左到右）

按钮 样子 作用 什么时候用
Logo 彩色图标 显示插件名称 -
共创 👥 按钮 联系开发团队 想提建议或反馈问题时
录制 ▶️ 或 ⏹️ 控制是否记录浏览 想暂停/继续记录时
设置 ⚙️ 齿轮 打开设置页面 修改语言、个人信息
头像 👤 登录/账号管理 登录获得更多额度
最小化 ⊖ 圆圈减号 隐藏面板，继续录制 想专心浏览但保持记录
关闭 ⊗ 圆圈叉号 关闭面板，停止录制 不需要记录时

3.2 中间主区域

这里会显示你的任务树。刚开始是空的，当你开始浏览网页后，这里会自动填充。

3.3 底部状态栏

元素 作用
🟢 绿点 + "AI 就绪" AI 连接正常，可以分析
🔴 红点 + "无 AI" AI 连接失败，点击可重试
⚡ 数字 今天消耗的 Token 数量
默认模式 / 监督模式 切换浏览模式（后面详细介绍）
AI 模式 / 离线模式 是否使用 AI 分析（建议保持 AI 模式）

📖 第四步：开始使用（核心教程）

4.1 正常浏览网页

  1. 确保插件面板已打开（点击工具栏图标）
  2. 确保录制状态为「开启」（顶部显示绿色方块 ⏹️）
  3. 像平时一样浏览网页

4.2 观察任务树的生成

当你访问一个新网页时：

  1. 插件会自动捕获这个页面
  2. AI 会分析这个页面属于什么任务
  3. 任务树中会自动更新

举例：
假设你在搜索 "如何学习 Python"，你可能会访问：

  • 百度搜索结果页
  • CSDN 教程文章
  • 知乎问答页面
  • B站视频页

插件会自动把这些页面归类到一个叫「学习 Python」的任务下。

4.3 理解任务树结构

📁 技术学习（领域）
   ├── 📋 学习 Python（任务）
   │   ├── 🌐 百度搜索：如何学习Python
   │   ├── 🌐 CSDN：Python入门教程
   │   └── 🌐 知乎：零基础学Python推荐
   │
   └── 📋 前端框架调研（另一个任务）
       ├── 🌐 React官网
       └── 🌐 Vue官网
  • 领域：大的分类（如：技术、购物、娱乐）
  • 任务：具体的事情（如：学习Python、买手机）
  • 页签：你访问的具体网页

🔍 第五步：查看和操作任务

5.1 展开/折叠任务

  • 点击任务前面的 ▶️ 箭头可以展开
  • 再次点击 ▼ 可以折叠

5.2 跳转到页面

  • 点击任何一个页签（网页）的标题
  • 浏览器会自动跳转到该页面

5.3 查看 AI 洞察

  • 把鼠标悬停在任务上
  • 会弹出一个小卡片，显示 AI 对这个任务的总结

5.4 生成分析报告

  1. 找到你想分析的任务或领域
  2. 点击右侧的 ✨ 分析按钮
  3. 等待几秒钟，AI 会生成一份详细报告
  4. 报告包含：
    • 这个任务的核心目的
    • 你的浏览路径分析
    • 关键发现和建议

5.5 关闭任务

  • 点击任务右侧的 ❌ 关闭按钮
  • 关闭的任务会进入「回收站」
  • 可以从底部的回收站恢复

🎯 第六步：使用监督模式（可选高级功能）

什么是监督模式？

当你想专注于某个任务，不被其他事情分心时，可以使用监督模式。

如何使用？

  1. 开启监督模式
    • 点击底部状态栏的「监督模式」按钮
    • 按钮会变成红色，表示已开启
  2. 设置主线任务
    • 在你想专注的领域或任务上
    • 右键点击 → 选择「标记为主线任务」
    • 该任务会显示 🎯 标记
  3. 专注浏览
    • 正常浏览与主线任务相关的网页
    • 如果你访问了无关的网页，插件会弹出提醒：「请专注于主线任务」
  4. 关闭监督模式
    • 再次点击「监督模式」按钮即可关闭

👤 第七步：登录与额度说明

为什么要登录？

  • 访客：每天只能分析 3 次
  • 登录用户：每天可以分析 20 次
  • Pro 用户：每天可以分析 200 次

如何登录？

  1. 点击顶部工具栏的 👤 头像按钮
  2. 在弹出的登录窗口中，选择登录方式
  3. 登录成功后，额度会自动增加

如何查看剩余额度？

  • 顶部的紫色横幅显示「今日额度 X / Y」
  • X = 已使用次数，Y = 总额度

🔧 常见问题解答

Q1: 为什么有些网页没有被记录？

可能的原因：

  1. 录制状态为「关闭」→ 点击 ▶️ 开启录制
  2. 今日额度已用完 → 等明天或登录获得更多额度
  3. 该页面不是 http/https 网页（如 chrome:// 页面不会记录）

Q2: AI 分析结果不准确怎么办？

解决方法：

  1. 在设置中更新你的职业和使用场景
  2. 这些信息越准确，AI 分析越准确

Q3: 我关闭了浏览器，数据还在吗？

答案：在！

  • 任务数据保存在本地存储中
  • 重新打开浏览器和插件，数据会自动恢复

Q4: 最小化后怎么恢复？

  1. 在当前网页的右下角，会出现一个小圆形悬浮球
  2. 悬浮球显示产品 Logo
  3. 绿点 = 正在录制，红点 = 已停止
  4. 点击悬浮球即可恢复插件面板

Q5: 如何切换语言？

  1. 点击 ⚙️ 设置按钮
  2. 在设置面板中找到「语言」选项
  3. 选择「中文」或「English」
  4. 保存设置

Q6: 如何完全重置插件？

  1. 进入 chrome://extensions
  2. 找到 IntentFlow 插件
  3. 点击「移除」
  4. 重新安装插件

💡 使用技巧

技巧 1：定期查看任务树

每隔一段时间看看任务树，回顾自己今天做了什么，有助于保持工作效率。

技巧 2：善用分析报告

完成一个任务后，生成分析报告，可以帮你总结收获和发现遗漏。

技巧 3：配合监督模式

当你需要专注工作时，开启监督模式，让插件帮你抵抗分心的诱惑。

技巧 4：登录账号

登录后不仅有更多额度，还能在多设备间同步（未来功能）。
